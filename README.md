# Hertz - Binaural Beats Generator

A desktop application for generating binaural beats with real-time visualization and session timer.

## Features

- 🎵 Generate binaural beats with customizable frequencies
- 📊 Real-time waveform and frequency visualization
- ⏱️ Built-in session timer
- 🌗 Dark/Light theme support
- 🔊 Volume control with fade in/out
- 💾 Preset frequency templates (Delta, Theta, Alpha, Beta, Gamma)
